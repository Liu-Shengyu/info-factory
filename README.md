# 信息工厂 · 30分钟闭环原型（GitHub Pages）

这是一个纯前端单页原型：探索 → 建线 → 稳定 → 放手（30回合≈30分钟体验闭环）。

## 如何用 GitHub Pages 打开
1. 把 `index.html` 放到仓库根目录（root）。
2. GitHub 网页端：Settings → Pages → Deploy from a branch
3. Branch 选 `main`，Folder 选 `/(root)`
4. 保存后等待 10–60 秒，访问：
   `https://<你的用户名>.github.io/<仓库名>/`

## 玩法（极简）
- 收件箱：读 / 暂存 / 忽略（消耗注意力）
- 5回合后解锁粗筛器（权重/去重/日报上限）
- 12回合后解锁升级（精度/缓冲/异常检测/手动介入）
- 20回合进入“缺席事件”，练习放手
- 25回合后回顾总结（可导出蓝图 JSON）

## 说明
- 本原型不依赖任何后端，不抓取真实信息，只用 A/B/C/D 抽象信号模拟策略—验证循环。
- 状态会写入 localStorage（同域名下保存）。
